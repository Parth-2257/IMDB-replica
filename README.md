# IMDb Clone

A React-based IMDb clone using TMDB API.

## Features
- Browse trending movies
- Search movies
- Filter by genre
- Movie detail page
- Watchlist
- Top rated
- Dark/light mode
- Responsive design

## Tech Stack
- React
- Vite
- React Router
- TMDB API
- CSS

## How to Run
1. **Install dependencies**:
   ```
   npm install
   ```
2. **Environment Setup**:
   Add your TMDB API key in a `.env` file as:
   ```env
   VITE_API_KEY=your_api_key_here
   ```
3. **Run the development server**:
   ```
   npm run dev
   ```
